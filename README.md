# My Price Alert App

This app compares product prices across Amazon, eBay, and Flipkart using official APIs.
It allows users to set alerts and view forecasts.

## Deployment
- Backend → Heroku
- Frontend → Vercel
- Database → Heroku Postgres
- Notifications → SendGrid + Firebase
